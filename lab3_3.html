Media Query Working Example.

Responsive Web Design uses HTML and CSS to resize websites to make them look good on different devices. This allows a programmer to include phones, tablets, etc into consideration for the creation of their website. This is done automatically and adjusts the screen sizes and viewports. To start using RWD, use meta name and set it equal to viewport
